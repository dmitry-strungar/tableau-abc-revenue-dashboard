# Дашборд: ABC-анализ выручки и структуры контрактов

## Обзор проекта

Данный проект представляет управленческий дашборд, созданный в Tableau для анализа структуры выручки компании и сегментации контрактов с использованием ABC-анализа.

Дашборд позволяет оценить:
- Распределение выручки по категориям услуг
- Вклад контрактов в общую выручку
- ABC-сегментацию услуг и контрактов
- Ключевые показатели эффективности (KPI)

---

## Компоненты дашборда

### 1. Блок KPI
- Общая выручка
- Количество контрактов
- Средний чек

### 2. Структура выручки по услугам
Гистограмма, отображающая долю выручки по категориям услуг.

### 3. ABC-анализ услуг
Кумулятивная кривая выручки с порогом 80%.

### 4. ABC-анализ контрактов
Ранжирование контрактов по вкладу в выручку:
- Сегмент A (первые 80%)
- Сегмент B (80–95%)
- Сегмент C (ниже 95%)

## Ключевые инсайты

### 1. Высокая концентрация выручки по типу услуг

Услуга **Рзработка ПО** формирует **62% общей выручки**, что делает её основным драйвером бизнеса.

Техническая поддержка генерирует около **36%**, в то время как консалтинг составляет лишь **~2%**, что указывает на выраженную специализацию компании.

Бизнес существенно зависит от одного направления.


### 2. Выраженный эффект Парето (ABC-анализ)

Сегмент **A** (верхние 80% накопленной выручки) включает ограниченное число контрактов, которые формируют основную долю дохода.

Накопительная кривая демонстрирует резкий рост на первых позициях, подтверждая сильную концентрацию дохода.

Это означает, что небольшое количество клиентов генерирует основную финансовую нагрузку бизнеса.


### 3. Дисбаланс контрактного портфеля

Сегмент **C** содержит большое количество контрактов с минимальной выручкой.

Их совокупный вклад незначителен по сравнению с операционными затратами на сопровождение.

Это может указывать на:

- неэффективную структуру контрактного портфеля  
- отсутствие минимального порога рентабельности  
- необходимость пересмотра ценовой политики  


### 4. Повышенный риск концентрации

Общая выручка составляет **21 804**, при этом значительная её часть сосредоточена в сегменте A.

Потеря даже одного крупного контракта может существенно повлиять на финансовый результат.

Модель бизнеса характеризуется высокой зависимостью от ограниченного числа клиентов.


### 5. Стратегический вывод

Компания имеет:

- сильную специализацию
- высокую концентрацию выручки
- потенциальный риск портфельной нестабильности

Для снижения риска возможны стратегии:

- диверсификация клиентской базы  
- развитие консалтингового направления  
- усиление кросс-продаж  
- перераспределение фокуса на сегмент B
